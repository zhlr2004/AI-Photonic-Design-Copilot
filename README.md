# AI Photonic Design Copilot

This repository contains Cursor skills for AI-assisted photonic design and paper reproduction workflows.

## Skills

- `paper-reproduction-prep`: prepares structured instructions for reproducing research papers.
- `UniParser-Paper-Markdown`: parses paper content into Markdown-oriented artifacts.
- `reproduce-instructor`: generates a simulation-grounded `instruction.md` from
  a PDF or a converted Markdown paper with sibling images. It supports
  API-driven open-source and commercial solvers without allowing analytical
  results to bypass construction and execution of the paper's model inside the
  selected simulator. The skill includes a reusable instruction template and a
  preferred simulator/library list with criteria for admitting other tools.
